功能一览

From quick CSV imports to enterprise-grade data pipelines.

3 Format Support

3 种格式支持

CSV, XLSX, and ODS files. OpenSpout-powered, memory-efficient streaming reader handles millions of rows.

支持 CSV、XLSX、ODS 文件。基于 OpenSpout 的内存友好流式读取，百万行无压力。

Auto Column Mapping

自动列映射

Fuzzy matching with Chinese/English aliases. Smart enough to figure out "姓名" → "name" automatically.

模糊匹配算法，内置中英文别名。自动识别"姓名"→"name"、"邮箱"→"email"等常见映射。

Row Validation

行级验证

Laravel Validator-powered. Define field rules, required fields, options, and defaults. Preview invalid rows before importing.

基于 Laravel Validator，支持字段规则、必填、选项列表、默认值。导入前即可预览无效行。

3 Duplicate Strategies

3 种重复策略

Skip, update, or error on duplicates. Detect duplicates across both existing DB records and incoming batch.

跳过、更新、或标记为错误。同时检测数据库中已有数据和本次批次内的重复。

Dark Mode

暗色模式

Full dark theme across all views. Merges seamlessly with your Filament dark-mode setup.

全部视图支持暗色模式，与 Filament 的暗色主题完美融合。

Multi-language

多语言

Built-in English and Simplified Chinese. Architecture ready for community translations.

内置英文和简体中文，架构支持社区贡献更多语言包。

Pipeline System Pro

管道系统 Pro

8 built-in pipes (TrimStrings, Lowercase, Bcrypt, DateFormat, DefaultValue, MergeColumns, SplitColumn, Closure). Field-level and global pipes.

8 个内置管道（去除空格、转小写、密码加密、日期格式化、默认值、列合并、列拆分、自定义闭包），支持全局和字段级管道。

Import History Pro

导入历史 Pro

Full CRUD resource with filters, sorting, stats widget, and bulk delete. Re-import failed/partial imports.

完整的 CRUD 资源页面，支持筛选、排序、统计 Widget、批量删除。可重新导入失败/部分成功的记录。

Queue & API Pro

队列与 API Pro

Auto-queue large imports. 5 REST API endpoints for integration with external systems.

大数据量自动转队列处理。5 个 REST API 端点，方便与外部系统集成。
